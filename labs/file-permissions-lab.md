# Lab: File Permissions in Linux

## Objective
Understand how file permissions work.

## Commands Used
- ls -l
- chmod 755 file.txt

## Steps
1. Checked permissions using ls -l  
2. Changed permissions using chmod  
3. Verified changes  

## Observations
- Permissions define access control
- Wrong permissions can expose data

## Key Learning
File permissions are critical for system security.

## Real-World Impact
Attackers can misuse weak permissions to gain access.
