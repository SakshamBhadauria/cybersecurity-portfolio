## 📘 Module 2: Network Operations

### TCP/IP Model (4 Layers)

1. Network Access → hardware + MAC  
2. Internet → IP, ICMP  
3. Transport → TCP, UDP  
4. Application → HTTP, DNS, SSH  

---

### TCP vs UDP

- TCP → Reliable, connection-based  
- UDP → Faster, no connection  

---

### Common Protocols & Ports

- HTTP → 80  
- HTTPS → 443  
- DNS → 53  
- SSH → 22  
- FTP → 21  

---

### Security Concepts

- Firewalls filter traffic  
- VPN encrypts communication  
- Network segmentation improves security  

---

## 📘 Module 3: Network Attacks

### Common Attacks

- **DoS / DDoS** → Flood network  
- **IP Spoofing** → Fake identity  
- **Replay Attack** → Reuse intercepted data  
- **On-path attack** → Intercept traffic  

---

### Packet Sniffing

- Passive → monitoring  
- Active → manipulation  

---

### Key Learning
Attackers exploit weak network security → monitoring & controls are critical

---

## 📘 Module 4: Security Hardening

### Key Concepts

- Security hardening = reducing vulnerabilities  
- Brute force attack = password guessing  

---

### Prevention Methods

- Strong passwords  
- MFA / 2FA  
- CAPTCHA  
- Limit login attempts  

---

### Security Tools

| Tool | Function |
|------|--------|
| Firewall | Controls traffic |
| IDS | Detects threats |
| IPS | Prevents attacks |
| SIEM | Monitors logs |

---

## 🎯 My Key Learning

- Networks are the foundation of cybersecurity  
- Understanding TCP/IP & OSI is critical  
- Most attacks happen through network vulnerabilities  
- Monitoring (SIEM, logs) is essential  
- Defense = layered approach (firewalls, IDS, policies)
